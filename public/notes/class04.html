CSS Color Property

• Text color can be set with color:


	h1 {
	  color: red;
	}

• Color values can be specified using:

⇒ color keyword: red

⇒ rgb: rgb(255, 0, 0)

⇒ hex code: #FF0000

CSS Color: Keyword Value

	span {
	  color: red;
	}

• Simplest; least-descriptive

• red, green, blue, etc.

⇒ deeppink

⇒ chartreuse

• More colors...

CSS Color: RGB Value

	span {
	  color: rgb(255, 0, 0);
	}

• Bit more complicated; bit more powerful

• First value is red; second is green; third is blue

• Each value in a range 0 - 255

• There are 16,777,216 possible choices.

• Values closer together are gray or muted:

rgb(200,200,200) / rgb(155,155,155) / rgb(075,075,075)

CSS Color: Hexadecimal Value

	p {
	  color: #463DB3;
	}

• Very robust...but there's some math happening

• Again there are 16,777,216 possible choices.

• How it works: "The Code Side of Color"

• Adobe Color: https://color.adobe.com

Applying Color: Text vs. Background

• color ⇒ Used to apply color to text


	p {
	  color: rgb(255, 0, 0);
	}

• background ⇒ Used to apply a background color (or image) to elements:


	p {
	  background: rgb(0, 255, 0);
	}

CSS Font: Various Properties

• font-weight

⇒ Sets the "thickness" of the font

⇒ normal and bold; values 100 - 900

• font-style

⇒ Sets a "slant" to the font

⇒ normal and italic

• font-size

⇒ Sets the size of the font

⇒ Pixel values (12px); percentage (75%); em's (1em)

CSS Font: Font-Family Property

• font-family

⇒ Applies a particular font to the element

⇒ Accepts multiple values

⇒ Should always end with: serif, sans-serif, or monospace


	p {
	  font-family: 'Comic Sans MS', helvetica, serif, sans-serif;
	}
CSS Font: Text Property

• text-align

⇒ how text is oriented within the containing block


	p {
		text-align: left; (or "right" or "center")
	}

• text-decoration

⇒ controls extra-textual visual emphasis (underline)


	p {
	  text-decoration: underline;

The Box Model

The Box Model

• Every HTML element is a box

• Each box is composed of:

⇒ content

⇒ padding

⇒ border

⇒ margin

• Awesome box model demo

The Box Model
The Box Model: Content

Content:

⇒ The text (and other elements) within the element's tags

The Box Model: Padding

Padding:

⇒ The space between the content and the border of the element

The Box Model: Border

Border:

⇒ The border around the element

The Box Model: Margin

Margin:

⇒ The space between the element's border and the other objects on the page

The Box Model: Borders

• Borders consist of three properties:

⇒ border-width: a pixel-based value for width

⇒ border-style: the style (solid, dotted, dashed, double)

⇒ border-color: the color

• Can apply all three values using the shorthand technique:


	p {
	  border: 2px solid black;
	}

	p {
	  border: rgb(255, 0, 0) 2px dashed;
	}
The Box Model: Padding

	// Method One:
	// targets each side individually
	p {
	  padding-top: 5px;
	  padding-bottom: 5px;
	  padding-right: 5px;
	  padding-left: 5px;
	}

	// Method Two:
  	// starts at top, moves clockwise: right, bottom, left
	p {
	  padding: 5px 5px 5px 5px;
	}

	// Method Three:
  	// padding: (top/bottom) (left/right)
	p {
	  padding: 5px 5px;
	}

	// Method Four:
	// will target all four sides (top, right, bottom, left)
	p {
	  padding: 5px;
	}
The Box Model: Margin

• Margin follows the same syntax as padding:


	p {
	  margin-top: 5px;
	  margin-bottom: 5px;
	  margin-right: 5px;
	  margin-left: 5px
	}

	p {
	  margin: 5px 5px 5px 5px;
	}

	p {
	  margin: 5px 5px;
	}

	p {
	  margin: 5px;
	}

The Display Property

The Display Property

• All HTML elements are either:

⇒ block

⇒ inline

• Controlled by the CSS property: display

⇒ display: block;

⇒ display: inline;

• A third option will be discussed later: inline-block

Block Elements

• Take up 100% of the width of their parent

• Placed below previous elements in the markup; break to a new line; stack like blocks

• Can assign margin and padding

• Can set height and width

Block Elements
Block Elements

• Examples of block elements:

⇒ <h1> thru <h6>

⇒ <p>

⇒ <ul>

⇒ <li>

Inline Elements

• Only take up the width of their content

• Do not send elements to the next line; renders inline, like this text

• Can assign padding

• Can only assign left and right margin

• CANNOT set height and width

Inline Elements
Inline Elements

• Examples of inline elements:

⇒ <a>

⇒ <strong>

⇒ <em>

<span> & <div>

<span> & <div>

• <span> is a generic inline element

• <div> is a generic block element

• THEY PROVIDE NO INHERENT MEANING

• Provide structure for styling other elements that they wrap

<span>

• The generic inline element

• <span> is used to add hooks onto specific inline words or phases within flowing content

<div>

• The generic block element

• <div> is used to organize other tags into blocks of content

• We use it for layout EVERYWHERE

A note on centering...

Centering Block and Inline Elements

• Center inline elements by setting the following on their parent:

text-align: center

• Center block elements by setting:

1) a width on the element (otherwise it will remain at 100% of its parent's width)

2) margin: 0 auto;
