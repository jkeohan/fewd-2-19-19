Basic Building Blocks

Basic Building Blocks

Basic Building Blocks Include:

  • svg (scaleable vector graphic)
    • - used for drawing elements such as: rect, circle, text
  • g (group)
    • - used to group svg elements
  • shapes (rect, circle, ellipse, line)
    • - used for things like bar charts, ect..

Must Know Methods

Must Know Methods For Creating Elements

Must Know Methods Include:

  • d3.select()
    • - used to select one element
  • d3.selectAll()
    • - used to select multiple elements
  • d3.append()
    • - used to append an element
  • d3.attr()
    • - used to add properties to an element

Step 1: Inital Setup

Step 1: Inital Setup

Initial Setup Includes:

  • Creating an SVG element
    • const svg = d3.select("body").append('svg').attr("width",1000).attr("height",400)
  • Append a g element
    • const g = svg.append('g').attr("transform","translate(50,200)")
  • And lastly we need some letters to play with.
    • const alphabet = "abcdefghijklmnopqrstuvwxyz".split("");

Step 2: Binding and rendering data

Step 2: Binding and rendering data

One of the most power set of tools that D3 offers is the ability to bind to data using .data()

  const text = g.selectAll("text").data(data,function(d){ return d;})

Now we use .enter() to render, position and style the data within a text element

  text.enter().append("text")
   .attr("x",function(d,i) { return i * 50;})
   .attr("y",0).text(function(d) { return d; })
   .attr("fill","green")
   .style("font-size","40px")
