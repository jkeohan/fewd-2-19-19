HTML Structure

HTML: History of HTML

• HTML => Hypertext Markup Language

• Majority of the websites you've visited are rendered in HTML

• Web browsers parse (read) HTML and render (display) the results

Example:Visit NYC

HTML Structure: <head>

• <head> : first tag inside <html></html>

• Adds additional, behind-the-scenes content

• Not displayed, but is machine parsable

• <title> : adds title to page

• <meta> : adds data...about data

• Some meta data: page description, keywords, author

• External CSS documents are linked in the <head>

HTML Structure: <body>

• <body> : second tag inside <html></html>

• Follows <head></head>

• Contains HTML/content that will be displayed to user

• All other HTML will be placed here

HTML Structure: Basic Structure

	<!DOCTYPE html>
	<html>
	  <head>
	    <title>The Website Title</title>
          </head>

	  <body>
	    <h1>ALL HTML CONTENT HERE!</h1>
	  </body>
	</html>

HTML

HTML Elements

• Fundamental building-block of HTML is the element

• (Most) Elements consist of:

⇒ Opening tag: <p>

⇒ Content: text, image, video, other elements, etc.

⇒ Closing tag: </p>

• The content is what the user sees

• The tags tell the browser what TYPE of content it is and how to present it

• Note: Some tags do not have a closing tag: <img>, <hr>, etc.

HTML Elements: Anatomy

• Important: the user will only see "Hello world!"

• The browser will apply some basic styling to <p> elements

HTML Tags Review

<h1></h1>
Heading Tags

• Used to display text as a title/headline of a webpage/section

• Tags <h1> through <h6>

• <h1> defines the most important title on the page

• Play a role in SEO; search engines pay special attention to what is in these tags

<h1></h1>

<h2></h2>

<h3></h3>

<h4></h4>

<p></p>
Paragraph Tags

• Used to group related chunks of text

• Like heading tags, browsers will apply default styling


  <p>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit.
    Dolores corporis perspiciatis amet autem sapiente fugiat sunt.
  </p>

  <p>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit.

    labore eum totam facere, assumenda

  </p>

<img> tags

<img> tags

• Images are placed using the <img> tag

• The <img> doesn't wrap content, so it doesn't close!

• Requires a src attribute; tells the browser where the source of the image (the image file) is located

• Both are valid:


	<img src="cookie.png">

	<img src="cookie.png"/>
src attribute

• The src attribute can be a relative/absolute path:

• Relative:


	<img src="cookie.png">

• Absolute:


	<img src="http://guardianlv.com/wp-content/uploads/2013/12/reeves2.jpg">
