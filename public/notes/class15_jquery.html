Review: Browser Events

• Events are actions that users perform in the browser:

⇒ click

⇒ mouse over

⇒ mouse out

⇒ key down

⇒ key up

⇒ form submit

• We can use JavaScript to track and respond to events

.onclick and .click()

• So far, we've primarily used the click event:

jQuery: .click()


	$('#someEl').click(function () {
	  // do some stuff
	})

JavaScript: .onclick


	document.querySelector('#someEl').onclick = function () {
	  // so some stuff
	}

<form> Element

<form> Element

• Forms are used to capture user input

• User input is sent to, and processed by, the server

• HTML forms can contain a variety of elements:

⇒ inputs and textareas

⇒ selects (dropdowns)

⇒ checkboxes

⇒ radio buttons

⇒ submit buttons

<form> Element

• Example form structure:


	<form>
		<input type="text" placeholder="Name">

		<select>
			<option value="one">One</option>
			<option value="one">Two</option>
		</select>

		<input type="checkbox"> Checkbox 1
		<input type="checkbox"> Checkbox 2

		<input type="submit" value="Submit Form">
	</form>

.submit() Event

.submit() Event

• Fires when <form> elements are submitted


	$('form').submit(function () {
	  // do something on form submit
	})

• On submit, will send form data to server (this does require configuration)

• Only works on <form> elements

• More info on <form> attributes

.submit() Event

• Specifically, .submit() will fire when:

⇒ User clicks <input type="submit>

⇒ User clicks <button type="submit></button>

⇒ User presses enter key while a <form> element is selected

Preventing Default Event Behaviors

Preventing Default Event Behavior

• Some events/elements have default behaviors

• For example, when clicked on, the following link will take you to http://google.com


	<a href="http://google.com">Google</a>

• Often, developers will want to override these default behaviors

Preventing Default Event Behavior

• Override defaults using .preventDefault()


	$('#myForm').submit(function (event) {
		event.preventDefault();

		// Rest of code here
	})

• The event argument can be passed into the anonymous function associated with the event

• You may also see this referred to as: evt, or even: e

Preventing Default Event Behavior

These all do the same thing:


	$('#myForm').submit(function (event) {
	  event.preventDefault();

	  // Rest of code here
	})



	$('#myForm').submit(function (evt) {
	  evt.preventDefault();

	  // Rest of code here
	})



	$('#myForm').submit(function (e) {
	  e.preventDefault();

	  // Rest of code here
	})
