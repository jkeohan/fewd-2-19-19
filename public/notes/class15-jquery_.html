Toggling Element Visibility

jQuery Hide and Show

.hide()

⇒ Hides elements by setting their inline style to display: none;


		$('p').hide();

.show()

⇒ Reveals elements by setting their inline style to display: block

⇒ If the element was originally inline, then display: inline will be used


		$('p').show();
jQuery Hide and Show

• Change the duration by passing an argument:

⇒ String:


	$('p').hide('slow')

	$('p').show('fast')

⇒ Number:


	$('p').hide(700)

	$('p').show(300)
jQuery Hide and Show

• jQuery provides alternatives to .hide() and .show():

.fadeIn()

.fadeOut()

.slideDown()

.slideUp()

.slideToggle()

• Each takes an optional duration argument:


	$('p').fadeIn('fast')

	$('p').slideUp(900)

Document Traversal: It's all in the family

.children()

⇒ Get an element's child elements

.parent()

⇒ Get an element's parent element

.siblings()

⇒ Get an element's sibling elements

Document Traversal: Siblings

.next()

⇒ Get the immediate following sibling element

.prev()

⇒ Get the immediate preceding sibling element

Document Traversal: Find
