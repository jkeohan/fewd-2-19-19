Reusable Scroller

Reusable Scroller

The scroller will do the following:

  • Given a set of sections it will figure where these elements are located vertically down the page.
  • When the page is scrolled it will figure out which of these elements is currently front-and-center in the browsers viewport.
  • It will make the new section the active element and initiate the visualization.

Initial Setup

Initial Setup

The two main data elements that we first need to grab in this project are:

  • The data points themselves, which we can import using the d3-request module or the content-specific convenience constructor methods such as: .csv(), .tsv(), .json()
  • All of the .step div elements which will be used to tell the story and step through the visualizations. We can grab these elements using d3.selectAll('.step')
